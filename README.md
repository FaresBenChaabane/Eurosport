# Eurosport
Technical Test for Eurosport Interview

- MinSdkVersion : 21
- CompileSdkVersion: 28
- TargetSdkVersion : 28

Features:
- Fetching remote articles.
- Sorting articles by date.
- Show articles mixed one-by-one(article-video-article-video.....).
- Show strories details.
- Stream videos.

Plus:
- 100 % Kotlin.
- Use of the Clean Architecture with a DDD (Domain Driven Development) Approach.
- Use of MVVM Pattern for Presentation Layer.
- Unit Tests for Data Layer/Domain Layer/Presentation Layer.

Waiting List (Not done yet):
- Offline Mode


NB : For Videos Streaming, make sure to use a phone and not an emulator. ExoPlayer from Google was used to stream videos.
